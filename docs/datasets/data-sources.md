# Data Sources

## Intended Source Set

- GDSC1
- GDSC2
- CCLE or DepMap drug sensitivity releases
- NCI-60 or CellMiner exports
- gCSI
- FIMM

## Required Documentation Fields

Each dataset page must include:
- source URL or publication
- version or release identifier
- retrieval date
- drug response metric used
- assay notes
- exclusion criteria
- known limitations

## Registry Dependencies

- Cellosaurus for cell line harmonization
- ChEMBL and PubChem for compound/target normalization
- indication source such as RepurposeDB or an explicitly named alternative
