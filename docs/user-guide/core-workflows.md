# Core Workflows

## 1. Explore Network Relationships

Goal:
Review drug-cell line-indication relationships through filters such as tissue type, target family, clinical phase, and source dataset.

Minimum documented evidence to add after code import:
- screen capture of filter selection
- screen capture of resulting network state
- description of exported artifact types

## 2. Export Publication-Quality Output

Goal:
Generate figures and tables that can support downstream analysis or manuscript preparation.

Required future documentation:
- output formats
- export resolution policy
- metadata included in exported tables

## 3. Run pIC50 Prediction

Goal:
Submit compound input through the site prediction flow and retrieve model output with version metadata.

Required future documentation:
- input schema
- output schema
- accepted cell line identifiers
- model version surfaced to users
- error and validation behavior
