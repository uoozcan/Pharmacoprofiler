# Platform Overview

Pharmacoprofiler is intended to support three core researcher tasks:

1. Explore harmonized pharmacogenomic relationships across drugs, cell lines, and disease context.
2. Export publication-ready figures and tabular summaries.
3. Request pIC50 predictions for compounds through the prediction workflow.

## Primary Views

- network exploration
- dataset filters
- compound and cell-line metadata panels
- prediction submission interface
- export controls
